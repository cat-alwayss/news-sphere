# 📰 NewsSphere

A modern full-stack News Application built using React, Vite, Express.js, and GNews API.

## 🌐 Live Demo

Frontend:
https://news-sphere-eight.vercel.app

Backend:
https://news-sphere-api.onrender.com

## ✨ Features

- 📰 Latest News
- 🔍 Search News
- 📂 News Categories
- ❤️ Reading List (Bookmarks)
- 🌙 Dark Mode
- 🌤️ Live Weather
- 📱 Fully Responsive Design

## 🛠️ Tech Stack

### Frontend

- React
- Vite
- Axios
- CSS

### Backend

- Node.js
- Express.js

### APIs

- GNews API
- OpenWeather API

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/cat-alwayss/news-sphere.git
```

Go into the project

```bash
cd news-sphere
```

Install frontend

```bash
cd client
npm install
```

Install backend

```bash
cd ../server
npm install
```

Start backend

```bash
npm run dev
```

Start frontend

```bash
cd ../client
npm run dev
```
